# An-English-vocabulary-instant-memorization-logging-app-for-simple-research
大学研究室での数週間の簡易研究のために開発した英単語即時記憶ロギングアプリです。エクセルを読み込んで単語学習の効果的なやり方を模索し、データを得ることができる。後戻りや一方通行の機能で、更に時間も決めることができる。その後はテストをしてその行動データをCSVで保存することが可能。
英単語即時記憶ログアプリ
フラッシュカード型 学習行動ログ収集システム
アプリ概要・使用方法・研究利用ガイド
1. 作成の背景と研究目的
本アプリは大学の探索的パイロット研究（参加者3名規模）のために開発されました。

研究タイトル（案）
「固定時間内フラッシュカード学習における行動ログ（滞在時間・訪問回数）を利用した英単語即時記憶予測モデルの探索」

研究の目的
•	学習中の行動ログ（各単語への滞在時間・訪問回数）が記憶定着にどう関係するかを探索する
•	自己調整学習（後戻り自由）の効果を定量的に検証し、ログから「この単語は覚えられているか」を予測する機械学習モデルの可能性を検討する
•	小規模（3人程度）の探索的パイロット研究として位置づけ、将来的な大規模実験への基盤とする

2. アプリ概要
本アプリは単一のHTMLファイル（index.html）で構成されており、サーバー不要でブラウザで直接開くだけで動作します。

ファイル形式	単一HTMLファイル（index.html）— サーバー不要
動作環境	モダンブラウザ（Chrome / Edge / Firefox / Safari）
インターネット	初回のみ必要（Excelライブラリ読み込みのため）。以降はオフラインで動作可
単語数	4〜50語（推奨10〜30語）。Excelまたは画面上で編集可

3. 全体の流れ
Step 1 ｜ 設定画面
•	参加者IDを入力（例：P01、P02、P03）
•	学習タイマーを設定（デフォルト10分、0.5分刻みで変更可）
•	学習モードを選択（後戻り自由 / Single Pass）
•	Excelテンプレートをダウンロードして単語リストを記入 → アプリに読み込む（またはアプリ上で直接編集）

Step 2 ｜ 学習フェーズ
•	フラッシュカード形式：表面＝英単語、クリックまたはスペースキーで裏面（日本語の意味）を表示
•	画面下の番号バーで現在位置・訪問済みのカードを視覚的に確認でき、クリックで直接ジャンプ可
•	←→矢印キー、前へ/次へボタンでカードを移動
•	カウントダウンタイマー表示（残り1分で赤く点滅）。タイマー切れで自動的にテストへ
•	「学習を終了してテストへ」ボタンで早期終了も可（終了理由もログに記録）

Step 3 ｜ テストフェーズ
•	全単語について英単語が出題され、正しい日本語の意味を四択で回答
•	出題順はランダム。誤答の選択肢は被りが少なくなるよう自動生成
•	各問の回答時間（秒）を自動記録

Step 4 ｜ 結果・ログ出力
•	正答率・正解数・実学習時間・総訪問回数のサマリーを表示
•	全単語の詳細ログを一覧表示
•	CSVダウンロードボタンでデータを保存（UTF-8 BOM付き、Excelでそのまま開ける）

4. 自動計測されるデータ（CSV出力列）

列名	内容	研究での利用
実施日時	セッション開始日時	識別・管理用
参加者ID	入力したID（例：P01）	個人別分析
モード	後戻り自由 / SinglePass	条件比較
設定タイマー（分）	設定した学習時間	実験条件記録
実学習時間（秒）	実際に学習した時間	条件制御
終了理由	タイマー終了 / 手動終了	条件制御
総滞在時間（秒）	0.3秒以上開いていた時間の合計	★主要特徴量（ML予測変数）
訪問回数	0.3秒以上開いた回数	★主要特徴量（ML予測変数）
裏面表示回数	意味を表示した回数	補助特徴量
出題順	テストで何問目に出題されたか	出題順序効果の分析
テスト正誤（0/1）	正解=1、不正解=0	★目的変数（予測ターゲット）
選択した回答	参加者が選んだ定義テキスト	誤答パターン分析
回答時間（秒）	選択肢が表示されてから回答までの時間	確信度の代替指標
全体正答率（%）	セッション全体の正答率	個人差・セッション比較

注：滞在時間は0.3秒未満のチラ見をカウントしない設計。タブを切り替えている間は計測を停止します。

5. 学習モードの説明

	後戻り自由モード	Single Passモード
概要	前後のカードに自由に移動できる	1枚ずつ順番に進み、後戻り不可
操作	矢印キー・ボタン・番号バーで任意移動	次へボタン・→キーのみ
研究での役割	介入条件（自己調整学習）	統制条件（比較対照）
測定ポイント	訪問回数・後戻り行動を重点計測	訪問回数は最大1（後戻りなし）

6. 単語リストの管理
Excelテンプレートを使った登録方法
•	設定画面の「Excelテンプレートをダウンロード」ボタンをクリック
•	ダウンロードした .xlsx ファイルを開き、A列に英単語・B列に日本語定義を記入（1行目はヘッダー行のため変更不要）
•	「Excel/CSVを読み込む」ボタンからファイルを選択 → リストが自動的に置き換わる
•	単語数は4〜50語に対応。読み込み後はブラウザに自動保存される

標準リスト（20語）について
アプリにはC1/C2レベルの英単語20語（ubiquitous, ephemeral, ameliorate ほか）が初期値として内蔵されています。「標準リストに戻す」ボタンでいつでも初期値に戻せます。

7. データ収集・分析の手順
実験ごとのデータ収集
•	参加者ごとにセッションを実施し、終了後に必ずCSVをダウンロードして保存
•	ファイル名：log_参加者ID_実施日時.csv（例：log_P01_2026-06-12-14-30-00.csv）
•	3人×2条件（後戻り自由・Single Pass）で計6ファイルを収集することを推奨

Excelでの分析
•	3人分のCSVを縦に結合するとそのまま分析データになる平坦な形式
•	列「モード」で条件を区別（後戻り自由 vs SinglePass）
•	列「テスト正誤（0/1）」が目的変数、「総滞在時間」「訪問回数」が主な説明変数

機械学習での予測モデル構築
•	特徴量：総滞在時間・訪問回数・裏面表示回数・回答時間（秒）・出題順 など
•	目的変数：テスト正誤（0/1）
•	小サンプルでも対応しやすい手法（ロジスティック回帰・決定木・ランダムフォレスト等）を推奨
•	参加者IDを特徴量に含める、または個人別モデルを構築することで個人差を考慮可能

8. 注意事項・制限
重要　Excelファイルの読み込みとテンプレート生成には、CDN上のライブラリ（SheetJS）を使用しています。初回利用時はインターネット接続が必要です。以降のカード学習・テスト・CSV出力はオフラインで動作します。
•	ブラウザのローカルストレージに単語リストが自動保存されます。別のPCや別のブラウザでは保存データが引き継がれません
•	同じブラウザで連続して実験を行う場合、前の参加者のログは保存されません。必ず各セッション後にCSVをダウンロードしてください
•	プレビュー版のため、学術論文への引用には動作確認・追加検証が必要です
